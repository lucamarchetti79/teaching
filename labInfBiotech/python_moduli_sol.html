Lezione 3.1 - Soluzioni¶

Esercizi¶

  1. Create un modulo che calcola i BMI.

  2. Create un modulo che stampi una frase.

Soluzioni¶

  1. Create un modulo che calcola i BMI.

    Aprite un editor di testo e scrivete il seguente codice salvandolo come file .py:

    peso = 50
altezza = 160
BMI = peso /(altezza * altezza)
print(BMI)

    Aprite una shell, raggiungete il percorso dove è stato salvato il file ed eseguitelo

  2. Create un modulo che stampi una frase.

    Aprite un editor di testo e scrivete il seguente codice salvandolo come file .py:

    testo = "Ciao come va"
print(testo)

    Aprite una shell, raggiungete il percorso dove è stato salvato il file ed eseguitelo
