Lezione 2.1 - Soluzioni

Esercizi

  1. Create 3 file usando l’editor nano o gedit

  2. Copiate i 3 file usando cp nel folder “prova”

  3. Create un quarto file chiamato concat.txt contenente la concatenzione dei 3 file creati

Soluzioni

  1. Create 3 file usando per esempio l’editor nano:

    nano a.txt
nano b.txt
nano c.txt

  2. Copiate i 3 file usando cp nel folder “prova”:

    mkdir prova
cp a.txt prova
cp b.txt prova
cp c.txt prova

  3. create un quarto file chiamato concat.txt contenente la concatenzione dei 3 file creati:

    cd prova
cat a.txt b.txt c.txt > concat.txt
